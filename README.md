  <h1>ForoHub - API REST con Spring Boot y Spring Security</h1>

  <p>ForoHub es una aplicación que simula un foro de discusión, implementando una API REST para la creación de tópicos. Utiliza las últimas tecnologías para ofrecer una solución eficiente y segura.</p>

  <h2>Características</h2>
  <ul>
    <li>Creación de tópicos mediante una API REST.</li>
    <li>Utilización de Postman para simular peticiones y pruebas.</li>
    <li>Implementación de Spring Boot para facilitar el desarrollo y despliegue.</li>
    <li>Integración de Spring Security con tokens JWT para asegurar las peticiones.</li>
    <li>Uso de MySQL como base de datos para almacenar la información de los tópicos.</li>
  </ul>

  <h2>Presentación</h2>
  <p>ForoHub utiliza un conjunto de tecnologías modernas para proporcionar una plataforma robusta y segura para la gestión de tópicos en un entorno de foro. A continuación se detallan las tecnologías clave utilizadas en el proyecto:</p>

  <ul>
    <li><strong>Java 22</strong>: Última versión del lenguaje Java, proporcionando las últimas características y mejoras de rendimiento.</li>
    <li><strong>Spring Boot</strong>: Framework de desarrollo para Java que facilita la creación rápida de aplicaciones basadas en Spring, simplificando la configuración y el despliegue.</li>
    <li><strong>Spring Security</strong>: Módulo de seguridad de Spring que proporciona autenticación y control de acceso a nivel de aplicación, integrando funcionalidades robustas de autenticación y autorización.</li>
    <li><strong>JSON Web Tokens (JWT)</strong>: Estándar abierto para la creación de tokens de acceso, permitiendo la autenticación segura y eficiente mediante la generación de tokens firmados digitalmente.</li>
    <li><strong>MySQL</strong>: Sistema de gestión de bases de datos relacional ampliamente utilizado, elegido por su capacidad para manejar grandes volúmenes de datos de manera eficiente y escalable.</li>
    <li><strong>Postman</strong>: Herramienta colaborativa de desarrollo de API que permite probar, desarrollar y documentar APIs de manera rápida y efectiva, facilitando el proceso de desarrollo y prueba de la API REST en ForoHub.</li>
  </ul>
